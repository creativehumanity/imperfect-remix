# Imperfect Remix
**A tool that reveals the human signature in music – imperfection not as flaw, but as fingerprint.**

Live demo: https://s78fxm.csb.app/

Built for Musixmatch Musicathon 2026 by Creative Humanity AI Hub.
creativehumanity.eu | team@creativehumanity.eu

## What it does
Measures human presence in music across 5 dimensions:
- Micro-Timing · Emotional Arc · Analog Texture · Lyrical Soul · Silence & Space

## APIs used
- Musixmatch Pro (lyrics + mood)
- LALAL.ai (stem separation)
- OpenAI Whisper (vocal transcription)
- Anthropic Claude (portraits)
- LetsSubmit (AI detection – coming soon)
